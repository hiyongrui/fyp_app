Tutorial

This page will be shown when there is no new template, click "New" to view how to add template

Language will change respectively for the symptoms and action. Click on the "+" button to add new symptom/action

Click on add symptom and action to add 1 symptom and 1 action

Add more action will be available after you have added symptom and action

When clicking on add action, you have to select the symptom you would like to add the action to

Once action added, popup will appear and you can select your desired action e.g. Call 995

This is how the final screen will look like, keep adding on as many symptoms and action as you like!

You can also long press symptom/action to delete them!
